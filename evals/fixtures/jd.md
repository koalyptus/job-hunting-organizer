# Senior Frontend Engineer — Reapit

**Location:** Remote (Australia)
**Type:** Full-time
**Salary:** 170k–190k AUD

## About Reapit

Reapit is a leading provider of property management software in the UK and Australia. We build tools that help real estate agencies manage their portfolios, from tenant onboarding to maintenance tracking. Our platform serves over 15,000 users across hundreds of agencies.

## The Role

We are looking for a Senior Frontend Engineer to join our product team. You will work on our flagship React application, building reusable components and improving the user experience for property managers. This role reports to the Engineering Manager and collaborates closely with Product and Design.

## Responsibilities

- Build and maintain reusable React components using TypeScript and Redux
- Collaborate with Design to implement accessible, responsive UIs
- Participate in code reviews and mentor junior developers
- Contribute to technical architecture decisions
- Write unit and integration tests using Vitest

## Requirements

- 5+ years of frontend development experience
- Strong TypeScript and React skills
- Experience with Redux or similar state management
- Familiarity with testing frameworks (Vitest, Jest, or similar)
- Excellent communication skills

## Nice to Have

- Experience with property management or real estate software
- Contributions to open-source projects
- Experience with AI-assisted development tools

## Tech Stack

- React 18, TypeScript, Redux Toolkit
- Vitest, Testing Library
- GitHub Actions, Docker
- PostgreSQL (for reference data)
